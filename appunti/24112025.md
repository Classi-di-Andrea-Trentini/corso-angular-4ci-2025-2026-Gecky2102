### Interfacce in TypeScript

In typeScript, un'interfaccia è una struttura che definisce il contratto per gli oggetti. Specifica quali proprietà e metodi un oggetto deve avere, senza implementare la logica.
Per creare un'interfaccia, si utilizza la parola chiave `interface`, seguita dal nome dell'interfaccia e dalle sue proprietà.
Esempio di creazione di un'interfaccia `IDocente`:

```bash
ng generate interface classi-interfacce/i-docente
```

Esempio di interfaccia `IDocente`:
```typescript
export interface IDocente {
    id: number;
    nome: string;
    cognome: string;
    materie: string[];
    classi: string[];
}
```
