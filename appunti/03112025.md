### Structural Directives
- Le direttive strutturali in Angular permettono di modificare la struttura del DOM aggiungendo o rimuovendo elementi in modo dinamico.

If
```TypeScript
@if(condition)
  <div>Questo elemento viene mostrato se la condizione è vera.</div>
@else
  <div>Questo elemento viene mostrato se la condizione è falsa.</div>
```
<br>

For
```TypeScript
@for(item of items(); track item) {
  <div>{{ item }}</div>
}
```
